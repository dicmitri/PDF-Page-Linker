# Privacy Notice This extension stores copied PDF page links locally in Chrome extension storage. Stored history entries contain: - PDF URL - Page number - PDF filename - Copied page link - Copy timestamp The extension does not send this history to any external server and does not include analytics or tracking code. PDF files are fetched from the URLs opened by the user so they can be rendered in the viewer. Deleting an entry from the popup removes that local history entry. Removing the extension or clearing its extension storage removes the stored history.
